# 估值分析

## 摘要
提供指数估值分析，包括 PE/PB 百分位、风险指标（回撤、波动率）。

## 用户故事
- 作为用户，我希望了解当前市场的估值水平
- 作为用户，我希望看到历史估值百分位

## 验收标准
- [ ] 支持 PE_TTM、PB、PS 等指标
- [ ] 历史百分位计算（排除负值和极端值）
- [ ] 风险指标：最大回撤、当前回撤、年化波动率
- [ ] 估值区间分类（低估/合理/高估）

## 边界条件
- 历史数据不足时显示"数据不足"
- 负 PE 不参与百分位计算
- 极端 PE (>1000) 不参与百分位计算
