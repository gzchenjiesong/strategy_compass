# 新闻聚合

## 摘要
从金十数据等来源聚合财经快讯，清洗后展示给用户。

## 用户故事
- 作为用户，我希望看到重要的财经快讯
- 作为用户，我希望快讯按重要性分类

## 验收标准
- [ ] 支持金十数据快讯接入
- [ ] 自动清洗 HTML 标签
- [ ] 重要性映射（高/中/低）
- [ ] 定时同步（5 分钟）
- [ ] 72 小时 TTL 自动清理

## 边界条件
- 数据源不可用时显示"暂无数据"
- 单条快讯长度限制 500 字
