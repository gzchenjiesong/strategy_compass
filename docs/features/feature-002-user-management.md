# 用户管理

## 摘要
管理用户基本信息、自选股、关注板块和偏好设置。

## 用户故事
- 作为用户，我希望管理我的自选股列表
- 作为用户，我希望关注感兴趣的板块
- 作为用户，我希望修改个人偏好设置

## 验收标准
- [ ] 用户可以添加/删除自选股（上限 100 只）
- [ ] 用户可以添加/删除关注板块
- [ ] 用户可以修改昵称、头像等基本信息
- [ ] 用户可以设置偏好（主题、通知等）

## 边界条件
- 自选股达到上限时提示"已达上限"
- 未登录用户无法访问用户管理功能
