# 数据底座

## 摘要
统一获取和管理外部金融数据（行情、估值、财务、新闻），提供标准化数据 API。

## 用户故事
- 作为系统，我需要统一的数据入口，避免各模块重复调用外部 API
- 作为用户，我希望看到实时、准确的金融数据

## 验收标准
- [ ] 支持 A股/港股/美股实时行情
- [ ] 支持指数估值数据（PE/PB/百分位）
- [ ] 支持新闻快讯聚合
- [ ] 数据缓存策略（内存 + SQLite）
- [ ] 首次部署自动初始化 10 个核心指数 20 年日 K

## 边界条件
- 外部 API 不可用时有降级策略
- 数据更新频率：行情实时，估值每日，新闻 5 分钟
