# 市场概览

## 摘要
提供 A股/港股/美股的市场概览，包括指数行情、估值分布、重要快讯。

## 用户故事
- 作为用户，我希望一眼看清市场整体状态
- 作为用户，我希望了解各指数的估值水平

## 验收标准
- [ ] 概览页：估值区间分布统计 + 核心指数卡片 + 重要快讯
- [ ] 大盘页：A股/港股/美股分组，详细指数卡片
- [ ] 指数卡片：价格、涨跌幅、估值、回撤
- [ ] 点击卡片弹出详情：K线图表 + 估值指标 + 风险指标

## 边界条件
- 无数据时显示"数据加载中"
- 支持响应式布局（桌面/平板/手机）
