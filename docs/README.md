# Strategy Compass 文档总览

> 本文档是项目文档的入口。所有 Agent 加载项目文档时，先读此文件了解文档结构。

## 文档分层

```
需求层 (features/)    — 做什么 (What)
设计层 (designs/)     — 怎么做 (How)
验收层 (acceptance/)  — 做得对不对 (Validation)
```

## 快速导航

| 目录 | 说明 | 适用角色 |
|------|------|----------|
| [features/](features/README.md) | 需求文档 | 产品经理 |
| [designs/](designs/README.md) | 模块设计文档 | 架构师、编码者 |
| [acceptance/](acceptance/README.md) | 验收报告 | 审查者 |

## 文档规范

- 单个文件不超过 2000 字
- 每个目录都有 README.md 作为入口
- 所有文档使用 YAML frontmatter 标记元数据
- 文档间通过相对路径链接关联
