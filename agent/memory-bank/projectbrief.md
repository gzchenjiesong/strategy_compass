# 项目概要 (Project Brief)

## 一句话说明

Strategy Compass（策略罗盘）—— 个人投资者的多策略投资管理平台。

## 目标用户

个人投资者，有以下特征：
- 有稳定收入和可投资资金
- 想让资产增值但没时间/精力盯盘
- 可能持有多只股票/基金，用不同策略管理
- 需要一个统一的仪表盘看全局

## 核心价值

1. **多策略统一管理** — 网格、定投、资产配置、红利管理，一个平台全看到
2. **数据驱动决策** — 用数据和指标辅助判断，而非靠感觉
3. **简洁直观** — 罗盘式的可视化，一眼看清方向和状态
4. **个人化** — 每个人的策略配置不同，系统适配你，不是你适配系统

## 范围

### 第一期（MVP）
- 网格交易引擎（从 codes-trading-py 迁移）
- 基础市场数据展示
- 用户认证（微信 OAuth）
- Web 前端仪表盘

### 第二期
- 基金定投策略
- 资产配置模块
- 财经资讯聚合

### 第三期
- 红利现金流管理
- 策略回测
- 多用户支持

## 技术约束

- 初期单机部署（腾讯云轻量服务器 4C8G）
- 数据库用 SQLite，后期按需迁 PostgreSQL
- 前后端分离，API 驱动
- 部署在自有域名 i-strategy-compass.com
