# 产品上下文 (Product Context)

## 解决什么问题

个人投资者面临的困境：

1. **信息过载** — 新闻、公告、研报、K线，信息太多不知道看什么
2. **策略分散** — 不同的股票用不同的策略，Excel 记得乱七八糟
3. **情绪干扰** — 涨了想追，跌了想跑，没有纪律性
4. **缺乏全局观** — 不知道自己整体的持仓分布、风险暴露、现金流状态

## 用户画像

**主要用户：** 有投资经验的个人投资者
- 年龄 25-45 岁
- 有编程基础或愿意学习
- 持有 5-30 只股票/基金
- 使用过至少一种主动策略（网格、定投等）

**次要用户：** 量化入门者
- 想尝试量化但不想写代码
- 需要可视化的策略管理工具

## 核心体验

1. **仪表盘** — 打开就看到：总资产、今日盈亏、策略运行状态、市场温度
2. **策略管理** — 每个策略一张卡片，状态一目了然（运行中/暂停/触发中）
3. **市场概览** — 核心指数、板块轮动、涨跌分布，不用开其他 App
4. **交易记录** — 每笔操作都有记录，支持复盘分析

## 竞品参考

| 产品 | 优势 | 不足 |
|------|------|------|
| 雪球 | 社区活跃、组合功能 | 不支持网格等自动化策略 |
| 同花顺 | 数据全面、功能丰富 | 太复杂，个人策略管理弱 |
| Excel | 灵活 | 无自动化、无可视化 |
| 自建脚本 | 完全可控 | 无 UI、维护成本高 |

Strategy Compass 的差异化：**专注于策略管理和执行，而不是行情或社交。**
