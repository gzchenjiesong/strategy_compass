## Module 1 设计确认项（2026-05-07）

### 数据源与策略决策

| 问题 | 结论 |
|------|------|
| 新闻数据源 | 使用**金十资讯**（j10data），MVP 阶段单一来源，未来增补 |
| 融资融券数据 | 沪深合计汇总级别，10 年历史，MVP 纳入 | 杠杆率百分位是市场热度核心指标 |
| 大盘指数历史数据 | **20 年**（上证、恒生、标普 500 等，看牛熊周期） |
| 行业/概念板块历史 | **15 年以上**（看行业规律） |
| 个股/宽基基金历史 | **10 年**（很多标的成立时间不足，按最长 10 年） |
| 数据粒度 | **日 K 为主**，至少周 K 级别 |
| 实时行情频率 | **分钟级以内**（非量化交易，操作按天进行） |
| 港股个股 | **MVP 支持**（港股通存在，数据必要） |
| 美股个股 | **MVP 后置**（先以功能实现为优先） |
