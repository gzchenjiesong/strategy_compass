## 今日成果：Phase 1 MVP 初版骨架完成

按照 `agent/encoding-prompt.md` 的编码阶段启动提示词，完成了 Strategy Compass MVP 的初版开发骨架。

### 后端（backend/）

- **Flask 应用工厂** + SQLAlchemy + SQLite(WAL) 配置
- **统一响应格式** + 自定义异常体系 + JWT 认证装饰器
- **Module 0 用户系统**：
  - 6 张数据表模型（users, invitation_codes, user_watchlists, watchlist_items, user_preferences, user_sector_favorites）
  - 微信 OAuth 回调（支持 mock 模式，无真实 AppID 时自动 mock）
  - 邀请码校验与使用次数追踪
  - 自选股 CRUD（单列表上限 100）
  - 板块关注 CRUD（上限 50，独立于自选股）
  - 用户偏好 Key-Value JSON 存储
- **Module 1 数据服务**：
  - 14 张数据表模型（指数/个股/板块 K线、估值、融资融券、新闻、同步日志）
  - 10 核心指数初始化脚本（含 A股6个 + 港股2个 + 美股2个）
  - 实时行情接口（AKShare + 腾讯财经 qt.gtimg.cn 备用）
  - K线查询接口（按 symbol + market + 日期范围）
- **Module 4 估值分析**：
  - PE/PB 历史百分位计算（过滤负值和极端值 >1000）
  - 风险指标：最大回撤、当前回撤、年化波动率
  - 估值区间标签：undervalued / neutral / overvalued
- **Module 2/3 骨架**：新闻列表 API、Dashboard 组装 API

### 前端（frontend/）

- **Vue 3 + Vite + TypeScript + Pinia + Tailwind CSS** 项目骨架
- Axios 封装（自动添加 Bearer Token，401 跳转登录）
- 底部导航栏（市场/资讯/关注/策略）
- **市场概览页面**：核心指数网格卡片 + 市场估值列表
- **资讯页面**：新闻列表 + 相对时间格式化
- **关注/策略页面**：占位页面
