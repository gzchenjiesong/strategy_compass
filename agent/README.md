# Agent 协作体系总览

> Strategy Compass 的 AI Agent 协作配置入口。

## 体系结构

```
memory-bank/    — 长期记忆（项目状态、架构模式、技术栈）
sub-agents/     — 角色定义（产品、架构、编码、审查）
specs/          — 技术规范（编码风格、API 设计、数据库）
sop/            — 操作流程（新功能、Bug 修复、部署）
```

## 快速导航

| 目录 | 说明 | 适用场景 |
|------|------|----------|
| [memory-bank/](memory-bank/README.md) | 项目长期记忆 | 了解项目现状 |
| [sub-agents/](sub-agents/README.md) | 角色定义与启动模板 | 开启新会话 |
| [specs/](specs/README.md) | 技术规范 | 编码和审查 |
| [sop/](sop/README.md) | 操作流程 | 执行任务 |

## 协作流水线

```
产品经理 → 架构师 → 编码者 → 审查者
   ↑________↓_________↓_________↓
   （反馈循环，问题可逐级或越级反馈）
```
