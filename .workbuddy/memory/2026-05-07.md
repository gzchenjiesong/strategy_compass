## Module 4 估值分析 — 前置研究

用户要求在进入 Module 4 设计前，先系统梳理金融行业中对不同标的类型的评估指标体系。

### 研究产出：四维度指标体系
1. **估值高低**：PE/PB/PS/PEG/PCF + 历史百分位（核心方法）
2. **热度指标**：涨跌幅排名、成交量、资金净流入、北向资金、ETF份额变化、融资余额等
3. **情绪指标**：涨跌家数比、涨停跌停比、波动率指数、融资买入占比、市场温度等
4. **回撤风险**：最大回撤、当前回撤、年化波动率、夏普比率、索提诺比率、卡玛比率、Beta等

### 关键结论
- **估值百分位是策略罗盘最核心的估值判断方法**
- 不同标的类型侧重不同估值指标：金融看PB、周期看PB+景气度、成长看PE+PEG、消费看PE
- 分析了分标的类型的指标矩阵（宽基指数/行业板块/概念板块/个股/ETF × 各指标维度）
- 待用户确认 MVP 范围后进入 Module 4 正式设计

## Module 4 估值分析 — 正式设计完成

产出文档：`docs/modules/module-4-valuation-analysis.md`

### API 接口（7 组）
1. **指数估值** — 单指数完整估值 + 批量指数估值（首页概览）
2. **板块估值** — 单板块估值 + 批量板块估值（排行+过滤）
3. **个股估值** — 完整估值分析（PE/PB/PS百分位 + 回撤 + 夏普 + Beta）
4. **ETF估值** — 通过关联指数获取 PE/PB 百分位 + 风险指标
5. **市场整体估值** — 杠杆率 + 融资余额百分位 + 北向资金
6. **估值区间统计** — 低估/适中/高估分布
7. **历史估值曲线** — 指数/个股 PE/PB 随时间变化（图表数据）

### 核心计算逻辑
- 百分位计算：PE/PB/PS/价格/融资余额/杠杆率，默认 10 年窗口
- 风险指标：最大回撤、当前回撤、年化波动率、夏普比率、卡玛比率、Beta
- 估值区间标签：≤30% 低估 / 30-70% 适中 / ≥70% 高估
- 性能优化：内存预计算 + 增量更新 + 1 小时缓存

### 设计决策
- 分标的类型指标体系（大盘/板块/个股/ETF/市场整体各不同）
- 百分位阈值 30%/70%，可配置
- Module 4 接口为内部调用，前端通过 Module 3/5/6 间接访问
- ETF 估值通过 ETF→指数映射表实现
